Current Project

“I’m developing Triforge TD — a mobile-first 3D tower defense game with merge-3 upgrading, hero abilities, and gacha progression.”

Built with Godot 4 (GDScript) for 768×1280 portrait. Features 8 tower types with ~86 augments, 6 hero classes, 17 enemy types across 63 campaign levels, world boss & challenge modes, and a full gacha economy. Backend uses PostgreSQL (Supabase), AWS Lambda + API Gateway (Python) for server-authoritative gacha with pity system, Firebase Auth, AdMob ads, Apple IAP + Google Play Billing + PayPal, and an offline-first architecture with cloud sync. Includes CI/CD via GitHub Actions, CDK infrastructure-as-code, and GDPR compliance.
